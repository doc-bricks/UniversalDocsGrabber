# Web/PWA-Companion

Stand: 2026-05-27

Dieser Ordner ist für einen späteren statischen Web/PWA-Companion vorgesehen.
Der Companion soll keine Mailkonten abrufen und keine privaten Dokumente
hochladen. Er dient nur zum Anzeigen und Bearbeiten redigierter Exporte aus der
Desktop-App.

## Geplante Funktionen

- Import von `docsgrabber-library-v1.json`
- Profil-, Kategorien- und Dokumentindex-Ansicht
- Suche und mobile Filter für Dokumentmetadaten
- Export aktualisierter Profilmetadaten zurück an die Desktop-App
- Android-/iOS-PWA-Smoke für Installierbarkeit, Offline-Start und Lesbarkeit

## Grenzen

- Kein IMAP-Abruf im Browser.
- Keine Speicherung von Passwörtern oder Tokens.
- Kein Upload privater PDFs oder Mailtexte.
- Keine native Android-/iOS-App, solange der PWA-Companion reicht.
